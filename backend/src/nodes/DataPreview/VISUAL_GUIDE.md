## Format Examples

### JSON (Pretty)
```
{
  "id": 1,
  "name": "John Doe",
  "email": "john@example.com",
  "active": true
}
```

### JSON (Compact)
```
{"id":1,"name":"John Doe","email":"john@example.com","active":true}
```

### Text
```
Counter: 5
Index: 4
Status: Active
```

### Table (Array of Objects)
```
+----+----------+-------------------+--------+
| id | name     | email             | active |
+----+----------+-------------------+--------+
| 1  | John Doe | john@example.com  | true   |
| 2  | Jane Doe | jane@example.com  | true   |
| 3  | Bob Lee  | bob@example.com   | false  |
+----+----------+-------------------+--------+
```
