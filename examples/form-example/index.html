nodeDrop Form Widget

Embed dynamic forms into any website with a single line of code

Auto-initialization

Simple Setup

Add the form widget to your page with just HTML attributes. No JavaScript required.

<div 
    data-nd-form="your-form-id"
    data-theme="auto"
></div>

<script src="nd-form-widget.umd.js"></script>
Manual Control

JavaScript API

Full programmatic control over the form widget lifecycle and configuration.

const widget = new nodeDropFormWidget();
widget.init({
    formId: 'your-form-id',
    theme: 'light',
    container: '#form-container'
});
